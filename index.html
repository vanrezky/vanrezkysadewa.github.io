PasarJempol is an online selling website. At PasarJempol you can order goods which you can directly checkout into What Apps or pay via Bank, this application is also equipped with postage calculations.

WsClient

WSClient is an application for synchronizing student and lecture data to the PDDIKTI feeder. This application is made very interactive so that the logs or data needed by the PDDIKTI Feeder can be known by the user during the synchronization process. The application has been used by large campuses such asUIN Suska Pekanbaru, Poltekpel Malahayati Aceh, Unitas Padang and others.
